## 📌 소개 (Introduction)

광운대학교 블록체인 학회 De-Butler는 차세대 인터넷의 주역이 될 인재들을 양성하고, 건전한 블록체인 생태계 발전에 기여합니다.
학회원 간의 원활한 교류와 연구, 개발 활동을 소개하기 위한 공식 랜딩 페이지입니다.

## 🚀 주요 기능 (Features)

- **Development Team**: Smart Contract 보안 취약점 분석, 블록체인 아키텍처 연구 및 DApp 프로토타입 개발 활동 소개.
- **Research Team**: L2 솔루션 생태계, 토큰 이코노믹스 및 DeFi 안전성 등에 대한 리서치 공유.
- **Activities**: 주간 세미나, 네트워킹 및 워크샵 경험 기록.

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: React (v19), TypeScript, Vite
- **Styling**: Tailwind CSS (v4)
- **Animation**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 💻 로컬에서 실행하기 (Run Locally)

**Prerequisites:** Node.js (v18 이상 권장)

1. 저장소 클론 및 폴더 이동:
   ```bash
   git clone <repository-url>
   cd de-butler-_-blockchain-academic-club
   ```

2. 패키지 설치:
   ```bash
   npm install
   ```

3. 로컬 서버 실행:
   ```bash
   npm run dev
   ```
